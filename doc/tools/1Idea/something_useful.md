### Bookmarks
book_mark is a wonderful tool when you read code with idea

- Toggle bookmarks
```$xslt
    ctrl+shift+number
    F11
    shift+ctrl+F11
```
- View bookmarks
```$xslt
#show bookmarks
shift + F11

# when you add bookmark by ctrl+shift+number
ctrl+number 

# custom shortcut key
# next
shift + ]
# previous
shift + [ 

```
